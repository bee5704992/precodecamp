valueของinput type ลักษณะ
button กำหนดปุ่มที่คลิกได้
checkbox กำหนดcheckboxให้เลือก
color กำหนดสีให้เลือก
date กำหนดให้เลือกวัน/เดือน/ปี(ไม่มีเวลา)
datetime-local กำหนดให้เลือกวัน/เดือน/ปี/เวลา(ไม่มีtimezone)
email กำหนดช่องเพื่อให้ใส่email
file กำหนดปุ่มให้คลิกเพื่อbrowseเลือกไฟล์
hidden กำหนดฟิลด์อินพุตที่ซ่อนอยู่
image กำหนดรูปให้คลิก(คล้ายๆปุ่ม)
month กำหนดเดือน/ปี(ไม่มีtimezone)
number กำหนดช่องเพื่อให้ใส่ตัวเลข
password กำหนดช่องเพื่อให้ใส่password(จะปิดบังสิ่งที่พิมพ์)
radio กำหนดchoiceให้เลือกแบบradio button
range กำหนดปุ่มให้ลากเพื่อcontrol range(เหมือนslider control)
reset กำหนดปุ่มให้รีเซ็ตค่า
search กำหนดช่องให้ใส่textเพื่อค้นหา
submit กำหนดปุ่มsubmit
tel กำหนดช่องให้ใส่หมายเลขโทรศัพท์
text กำหนดช่องให้ใส่ข้อความ
time กำหนดการใส่เวลา(ไม่มีtimezone)
url กำหนดช่องเพื่อให้ใส่url
week กำหนดสัปดาห์และปี โดยจะบอกว่าสัปดาห์ที่เท่าไร มีวันอะไรบ้าง และอยู่เดือนไหน (ไม่มีtimezone)
