tag คำอธิบาย
strong กำหนดให้ข้อความเป็นตัวหนา(เหมือนแท็กb)
label ข้อความไม่เปลี่ยนแปลง เพราะlabelไว้ใช้กับelementsอื่นๆเช่น input type
del ขีดฆ่าที่ข้อความ
b กำหนดให้ข้อความเป็นตัวหนา
em กำหนดให้ตัวอักษรเอียง(เหมือนแท็กi)
mark ทำมาร์ค/ไฮไลท์ข้อความ
i กำหนดให้ตัวอักษรเอียง
sub ทำให้ข้อความตัวเล็กลงอยู่ข้างล่าง เช่น เลขของธาตุH2O
sup ทำให้ข้อความตัวเล็กอยู่ข้างบน เช่น เลขยกกำลัง
